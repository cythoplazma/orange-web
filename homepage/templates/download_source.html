Linux / Source

Installing with Anaconda

If you are using python provided by Anaconda distribution, you are almost ready to go. Just run:

conda install orange3

Installing with pip

The pip package depends on a PyQt4 system installation. Use your linux distribution's package manager to install it first. Then, Orange can be installed with: 

pip install orange3

Installing from source

Clone our repository from GitHub or download the source code tarball. Then follow the instructions in README.md

Running Orange from command line

To run Orange Canvas run

python -m Orange.canvas
