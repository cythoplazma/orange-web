Visual Programming

Orange is a great data mining tool for complete beginners as well as for expert data scientists. Its graphic user interface allows the users to focus on data analysis instead on laborious coding, making a construction of complex data analytics workflows extremely simple. Most of Orange's visualisations are interactive and support data selection and filtering.

Component Based Software

You can easily perform data analysis by stacking components into schematic workflows. Each component embeds some data retrieval, preprocessing, visualisation, modelling or evaluation step in your data research. Combining different components (widgets) in a workflow enables you to build comprehensive data analysis schemas as you go. With a large library of preloaded widgets you won't be short for choice. Additional widgets are available through add-ons and allow for a more focused and topic-oriented research.

Interactive Data Exploration

Orange widgets communicate with each other. They receive data on the input and send out filtered or processed data, models, or anything the widget does on the output. Connect the output to another widget and you have a functioning workflow. Alter the settings or select a data subset in one widget and instantaneously see how the changes affect other widgets. Through the choice of the right widgets and their connections, it is easy to construct complex workflows for a broad variety of data analysis tasks.

Clever Design Interface

Orange is easy to use even for complete novices. Start with the File widget and Orange will automatically suggest the next widgets that can be connected to it. For example, Orange knows you are likely to want Hierarchical Clustering after you've set up your Distances widget. All other defaults in the widgets are also set in a way that enables a simple analysis even without knowing a whole lot about statistics, machine learning, or exploratory data mining in general.
